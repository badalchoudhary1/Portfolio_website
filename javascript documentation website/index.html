Introduction

Javascript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host enviroment (for example, a web browser), Javascript can be connected to the object of its enviroment to provide programmatic control over them.

Javascript contains a standard libraray of objects, such as array, Date and Math,and a core set of language elements such as operators, control structures , and statements. Core Javascript can be exented for a variety of purpose by supplementing it with additional for a variety of purpose by supplementing it with additional objects;for example

  • Client-side Javascript extends the core language by supplying objects to control a browser and its Document object model(dom) for example clientside extension allow an application to place elements on an html form and respond to user events such as mouse clicks, form input and page navigation.
  • Server-side Javascript extends the core langyuage by supplying objects relevent to running Javascript on a server for example server side extension allow an application to communite with a database, provide continuity of information from invocation to another of the application , or application or perform file manipulation on a server.
